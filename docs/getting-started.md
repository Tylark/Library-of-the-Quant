# Getting Started

Library of the Quant is organized around finished educational PDF manuals and companion Markdown lookup notes.

## Where Things Live

- Finished PDFs: `manuals/pdf/`
- Companion Markdown notes: `manuals/markdown/`
- Library indexes: `indexes/`
- Project documentation: `docs/`

## How To Read The Library

Use the master index for the full catalog, the topic and difficulty indexes for guided study, the formula index for reference lookup, the implementation traps index for practical research warnings, and the system design index for architecture-oriented concepts.

All material is educational only and is not financial advice.
